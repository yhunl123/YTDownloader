직관적인 GUI 환경에서 고화질 영상과 음원(MP3)을 손쉽게 다운로드하고 관리할 수 있습니다.

## 1. 개요 (Overview)

이 프로젝트는 사용자가 유튜브 링크를 입력하면 해당 영상을 분석하여 원하는 포맷(MP4, MKV, MP3)과 화질(최고 화질 ~ 360p)로 다운로드하는 데스크톱 애플리케이션입니다.

### 주요 특징
* **다양한 형식 지원:** 일반 영상 URL, 단축 URL(youtu.be), 쇼츠(Shorts) 완벽 지원.
* **고화질 다운로드:** 1080p 이상의 고해상도 영상과 오디오를 병합하여 저장 (FFmpeg 사용).
* **스마트한 UI:**
    * 다운로드 진행 상황(프로그레스 바)과 영상 정보(썸네일, 제목, 길이, 용량)를 실시간 표시.
    * 음각(Sunken) 스타일의 리스트 뷰와 가독성 높은 디자인.
* **편의 기능:**
    * 프로그램 종료 후 재시작 시 다운로드 리스트 및 설정(경로, 옵션) 자동 복구 (세션 저장).
    * 우클릭 메뉴를 통한 파일 위치 열기, URL 복사, 삭제 기능.
    * 중복 다운로드 방지 로직 탑재.

## 2. 설치 및 사용법 (Installation & Usage)

### 사전 준비 (Prerequisites)
이 프로그램은 **Python 3.x** 환경에서 실행됩니다. 또한 고화질 영상 병합 및 MP3 변환을 위해 **FFmpeg**가 필수적으로 필요합니다.

### 1. 라이브러리 설치
터미널(CMD)에서 아래 명령어를 입력하여 필요한 패키지를 설치합니다.

```bash
pip install PyQt5 yt-dlp requests
```

### 2. FFmpeg 설치 (필수)
유튜브의 고화질 영상은 비디오와 오디오가 분리되어 전송되므로, 이를 합치기 위해 FFmpeg가 필요합니다.
1.  [FFmpeg 다운로드 페이지](https://www.gyan.dev/ffmpeg/builds/) 접속.
2.  `ffmpeg-release-essentials.zip` 다운로드 및 압축 해제.
3.  `bin` 폴더 안에 있는 `ffmpeg.exe` 파일을 **이 프로그램의 `main.py`가 있는 폴더**에 복사해 넣습니다.

### 3. 프로그램 실행
```bash
python main.py
```

### 4. 사용 방법
1.  **URL 입력:** 상단 입력창에 유튜브 링크를 붙여넣고 `Enter` 키 또는 `입력` 버튼 클릭.
2.  **옵션 설정:** 파일 형식(mp4, mkv, mp3)과 화질을 선택 (다운로드 중에도 변경 가능).
3.  **경로 지정:** `찾기` 버튼으로 저장할 폴더 선택 (기본값: `./download`).
4.  **관리:** 리스트의 항목을 우클릭하여 폴더 열기, 삭제, 재시도 등을 수행.

---

## 3. 주의사항 (Precautions)

1.  **저작권 준수:** 다운로드한 영상은 반드시 **개인 소장용**으로만 사용해야 하며, 무단 배포나 상업적 이용 시 저작권법에 저촉될 수 있습니다.
2.  **FFmpeg 오류:** 다운로드 중 "변환 및 저장 중..." 단계에서 멈추거나 에러가 발생한다면 `ffmpeg.exe`가 프로젝트 폴더에 있는지 확인해주세요.
3.  **동시 다운로드:** 너무 많은 영상을 동시에 다운로드하면 시스템 리소스(CPU, 네트워크)를 많이 점유할 수 있습니다.
4.  **중복 방지:** 이미 리스트에 존재하고 완료되지 않은 영상은 중복해서 추가할 수 없습니다. (충돌 방지)

### v1.3 (편의성 증대)
